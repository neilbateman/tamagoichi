# Tamagoichi

#### By **Neil Bateman**

### Description
* _A digital pet you must take care of, or it will die._

### Setup/Installation Requirements
* _Clone repository and run 'npm install' thru terminal._
* _From terminal, in project directory, run 'npm run start'._
